Similarity between Subreddits

This bar chart shows the similarity between randomly chosen subreddits to the subreddit 'news'.
Similarity is calculated by pearson correlation (scroll down) using the percentage of posts in each category for each subreddit as the value for calculation. The category of each post is determined by looking up Open DNS. We count the number of posts in each category and use the percentage of posts from each category to calculate the pearson correlation score, which determines whether one subreddit is similar to another subreddit.
